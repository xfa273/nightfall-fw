## Mode 2-7: 最短走行

共通動作:

- case 選択後、保存済みマップから `solver_build_path(mode, case)` で経路生成。
- case ごとの直線速度/加速度を適用。
- mode ごとの小回り/大回り/斜めターン速度と fan を適用。
- `drive_fan(fan_power)` 後に `run()`。

経路タイプ:

- case1: 小回り中心。`path_type=0`
- case2-7: 大回り変換あり。`path_type=1`
- case8-9: 斜め変換あり。`path_type=2`
