## マイクロマウスの運び方

機体は自宅・活動場所・大会会場の間で持ち運ぶ必要がありますが，精密機器なので雑に運ぶと破損や再調整が発生します．
100均ケースやタッパー運用もありますが，本記事では「機体と周辺機器をぴったり収納できる専用ケース」を作る方針です．

## やりたいこと

ケースに欲しい条件は以下です。

- そこそこ頑丈で，ケース単体で持ち運べる
- 機体と周辺機器（バッテリー，書き込みケーブル，掃除用テープ）が入る
- ケース内で機体や小物が動かない

## 買ったもの

- 小型アルミケース
- 切れ込み入りスポンジブロック

元記事ではブラックフライデー価格で合計約2,700円との記載です。

## 作り方

1. ケース内の仕切りを外す
2. 内寸に合わせてスポンジを切ってはめる
3. 機体・周辺機器がぴったり入る切り欠きを作る
4. スポンジを一度外して両面テープで固定する
5. 高さ合わせのため底面にスポンジ片を追加する
6. 必要に応じてフタ側にマジックテープを貼る

## まとめ

小型アルミケースと切れ込み入りスポンジの組み合わせで，実用的なマイクロマウス用キャリングケースが作れるという内容です．
輸送・保管で機体を壊すと開発リズムが崩れるため，運搬系の治具を先に整えるメリットは大きいという結論です。
