## モーターマウント

モーターマウントは各自の機体設計に合わせて設計しますが，以下を満たすようにします．

- 車軸の固定部は `M3` めねじ
- 車軸固定部に直径`4.1mm`，深さ`1mm`のざぐりを作る

ねじ切りが必要なので，材質はアルミかPOM切削がおすすめです．
3Dプリントの場合は，反対側にナットを埋め込んで代用しても良いと思います．

## 全体像

車体の外側から順に，

`モーターマウント ← スペーサー(1.6mm) ← ベアリング ← ギヤ ← ホイール ← スペーサー(1mm) ← ベアリング ← ねじ`

です．
ベアリングの内輪はスペーサー経由でモーターマウントに締め付けられて固定され，
ベアリングの外輪がギヤホイールとともに回転します．

## 組立て

まず，ホイールの穴をテーパリーマで広げます．
中の段差がギリギリなくならないぐらいまで削ります．

そして，ホイールのオフセット部分にスパーギヤをはめ込みます．

次に，スペーサーを長さ`1.6mm`から`1mm`（プラス公差）に削ります．
モータマウントの軸受取り付け穴のざぐり部分にスペーサーをはめ込み，
やすりでマウント面とツライチになるまで削ると，ざぐり深さが`1mm`なのでスペーサーも`1mm`にできます．

ベアリング内輪を支えるために長さは`1mm`よりわずかに長い方がよく，`1.08mm`くらいが良さそうです．
でき上がったスペーサーは外して次の手順で使います．

そして，削ったスペーサーを使って，
`ねじ - ベアリング - スペーサー` をホイールに差し込みます．
反対側にもベアリングをはめ込み，ねじを通します．

最後に，モーターマウントのざぐり部に今度は削っていないスペーサーをはめ込んでから，
組み立てたギヤホイールをねじで固定します．
スペーサー長さが適切なら，強く締め付けてもホイールの回転は滑らかなままです．

これで完成です．

## 補足

ホイールは樹脂製でも使えますが，
ホイールとギヤのはめ込み部分に強い負荷がかかったときに割れることがありました．
通常走行では起きにくい気もしますが，安心のためにはアルミホイールがおすすめです．
