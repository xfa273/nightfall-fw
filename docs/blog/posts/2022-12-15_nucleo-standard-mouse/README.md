## 0. お詫びとはじめに

本記事は本来の公開時刻より遅れて投稿されたことへのお詫びから始まります。
そのうえで，WMMC標準マウスを「より作りやすくする」目的で，Nucleoベースの派生機体を試作した経緯を紹介しています。

## 1. WMMCの標準マウスって意外と作るのキツい...？？

標準マウスは実績ある構成ですが，初心者視点では以下が難所になりがち，という整理です。

- ユニバーサル基板のはんだ付け難度
- センサ基板の実装密度
- 多ピンケーブルのカシメ
- 基板・底板固定の再現性
- センサ光軸合わせ

## 2. 簡単のための仕様を考える

改善方針として次を設定しています。

- 全てプリント基板
- 表面実装なし
- カシメが必要なケーブルを最小化
- パーツ数の削減
- 光軸合わせの簡略化
- 標準マウス部品を可能な限り流用

## 3. 機体構成

### メインボード

- マイコン: 石のSTM32F303K8T6 から Nucleo32 に変更
- モータドライバ: 標準マウス同等（SLA7078MPRT）
- 電源: Li-Po 3S → 5V DC/DC → Nucleo 5V 入力
- インターフェース: 3スイッチ + 3LED（追加LED2個）

### センサボード

標準マウス相当回路を維持しつつ，メイン基板切り欠きへ直接差し込む構成にして，光軸合わせと配線を簡略化しています。

### 底板

5mmアクリルではなく，虚無PCB（配線なし基板）を底板として使う案を採用。

## 4. 取り敢えず出来上がった機体

実際に組んだ機体では，製作性は狙い通り改善され，短時間で動作確認まで到達できたとしています。
また，試作機でスラローム・最短走行まで確認し，入門機としては十分な性能感という評価です。

## 5. 既知の問題点と改良ポイント

元記事で挙げられている主な課題:

- Nucleo裏のジャンパ抵抗除去が必要になる点
- Enableピン仕様起因の不要励磁
- 電源コネクタ仕様（XA）と運用性
- 一部シルク誤記
