# hardware/mini_r1_0

現行 mini 機体 (`mini_r1_0`) のハードウェア情報置き場です。

## 構成

- `cad/`: 回路図・基板データ
- `bom/`: 部品表・調達情報
- `notes/`: 調整メモ・確認記録

必要な実データは今後ここへ集約します。

## 備考

- STM32F405 搭載ハーフサイズの最終版です。
- 対応する Eagle プロジェクトは `HM_Nightfall-mini_v1` です。
- `cad/eagle/` に Eagle 移行前原本を保存しています。
- Eagle 9.6.2 で生成した参照アーカイブは `cad/export/reference/eagle_archive_20260606/` に保存しています。
