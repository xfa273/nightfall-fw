# hardware/classic_r1_0

現行 classic 機体 (`classic_r1_0`) のハードウェア情報置き場です。

## 構成

- `cad/`: 回路図・基板データ
- `bom/`: 部品表・調達情報
- `notes/`: 調整メモ・確認記録

必要な実データは今後ここへ集約します。

## 備考

- STM32F405 搭載クラシックサイズの最終版です。
- 対応する Eagle プロジェクトは `CM_Nightfall-Air_v1` です。
- `cad/eagle/` に Eagle 移行前原本を保存しています。
- OneDrive 側の発注用 zip は未取り込みです。必要になった発注版だけ `cad/export/` または GitHub Release で管理します。
